Beskrivning
Nu ska du skapa en layout för ett projekt som skall bli en webbshopp med react och react router.

Kravet från kund är att det skall finnas en startsida, en kontaktsida samt en produktsida med möjlighet att ha en undermeny (produkter).

Kunden driver företaget Sneaky Sneakers och har bifogat lite material för att hjälpa dig komma igång!

Bonus, importera json filen och skapa produktsidan med produkt-routern dynamiskt med map().
